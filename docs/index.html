How to get started creating awesome looking quizzes easily and quickly using jQuizzy

What is jQuizzy

As you probably know at this point, jQuizzy is a simple but powerful quiz engine that lets you create interactive quizzes. The default interface is attractive, intuitive, sleek and more importantly, completely customizable.

The engine has a number of nifty features like:

  • Progress bar
  • Moving back and forth between items
  • Question counter
  • Reviewing results at the end of the quiz
  • Social sharing via Twitter

Of course, there are just the main points of this release -- there are a number of small features that you'll find throughout the codebase and interface.

I'll be listening to the feedback from buyers and adding features to future versions.

 

Getting Started

Getting up and running with jQuizzy is incredibly easy. Let me walk you through the process.

Step 1

First step is to create a DIV element for our plugin. This element will be the container for all the quiz related markup that the plugin will generate.

<div id='quiz-container'></div>

Step 2

Next we'll need to include the stylesheet for jQuizzy and the jQuery library, if you're not already using it.

<link rel='stylesheet' href='css/styles.css' /><script src='js/jquery.js'></script>

Step 3

Include the actual jQuizzy plugin. The source contains 2 version of jQuizzy -- a compressed, minified version for deployment and a normal version for development and testing.

<script src='js/jquery.jquizzy-min.js'></script>

Step 4

Create the init object to hold all our questions. Use the following code to create the object.

var init = {};

Step 5

Now, you can populate the object with your questions and the expected answer. Like so:

var init = {'questions': [{'question': 'jQuery is a...', 'answers': ['JavaScript library','Ruby Gem','PHP Framework','None of the above'],'correctAnswer': 1},{'question': 'X comes after?','answers': ['P','W','Y','v'],'correctAnswer': 2},{'question': 'Meh stands for..','answers': ['Eh','/Sigh','What?','Whatever'],'correctAnswer': 4}]};

The syntax and structure is fairly self explanatory.

Please make sure to escape quotes when defining the questions and answers. You can do it by prefacing each quote with a backslash. So instead of let's you'd use let\'s.

WordPress and other CMS systems sometimes break the formatting of textual content by adding <p> and <br /> elements. In these cases, please move the init object to its own JavaScript file and then include it after the jQuizzy plugin.

Step 6

Now you can initialize the jQuizzy plugin. Here's how:

$('#quiz-container').jquizzy({            questions: init.questions,        });


That's it! You now have a completely functional, slick looking quiz running.

 

Plugin Options

jQuizzy is fully customizable by providing options to change various settings and options throughout the quiz and the quiz engine. Here is a list of the settings that you can change:

questions - Object

This is the most important option of them all. If you fail to pass in an option, your computer will explode and the sky will turn red. Haha! Nah, if you fail to pass this option, the entire engine will stall with an error message.

Please remember to follow the overall structure for the questions object as shown in the 'Getting Started' section of the documentation.

twitterStatus - String

A string containing the status you'd like to post to Twitter. You can use the {score} modifer to place the score within the sentence. For example,

Yay! I scored {score} on this awesome quiz!

It's wise to include a shortened link so the Twitter status makes more sense. By default, this is the status set in the engine:

Scored {score} on this awesome quiz! Check it out! http://linky.com

startText - String

The text that's displayed at the top right corner of the engine when the quiz first loads. The default is Let\'s get started

endText - String

The text that's displayed at the top right corner of the engine when the quiz has ended. The default is Finished!

splashImage - String

Image to display as the splash or main image when the quiz loads. The default is img/start.png

twitterImage - String

Image to display as the link to Twitter sharing on the final part of the quiz. The default is img/share.png

resultComments - Object

Using this option, you can pass in overrides for the comments that are displayed during the end of the quiz. For example, if you run a quiz for a fan site of some sort, you can tailor it to your market.

If you're overriding this, please make sure to follow the exact syntax for the resultComments object.

resultComments :        {            perfect: 'Perfect!',             excellent: 'Excellent!',             good: 'Good!',             average: 'Acceptable!',             bad: 'Disappointing!',             poor: 'Poor!',             worst: 'Nada!'      }

Again, should be fairly self explanatory. Just replace the second part of the line with your text and you're all set.

These are all the user exposed options right now but based on user feedback I'll look into opening up more of the engine to the end user.

Customizing the UI

The jQuizzy engine uses a simple CSS stylesheet to handle the presentation of the engine. You can find this stylesheet under css/styles.

Please don't tinker with the CSS unless you know what you're doing. The CSS is paramount to the engine's presentation and I can't be held liable if you try customizing it and the engine breaks.
